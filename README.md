# 다형성 좀비 콘솔 게임

> ### 소개
>
히어로가 되어 좀비와 보스를 무찌르는 게임 <br>
앞으로 이동하며 좀비, 보스를 만나면 전투를 하여 승리하면 이기는 게임 <br>
히어로는 공격하기와 포션마시기를 선택 할 수 있으며, 좀비와 보스를 둘다
무찌르면 게임 클리어 <br>
히어로가 죽으면 게임에서 패배
